📱 Harcama Takip Uygulaması (Ionic & Angular)
Bu proje, mobil programlama dersi kapsamında geliştirilmiş, kullanıcıların günlük harcamalarını kategorize ederek takip etmelerini sağlayan bir Ionic uygulamasıdır.

✨ Özellikler

-- Harcama Ekleme: Harcama adı, tutarı ve kategorisi ile sisteme kayıt yapılır.

-- Dinamik Listeleme: Eklenen harcamalar anlık olarak listede görüntülenir.

-- Toplam Hesaplama: Tüm harcamaların toplam tutarı otomatik olarak hesaplanır ve TRY formatında gösterilir.

-- Veri Kalıcılığı (LocalStorage): Uygulama kapatılsa veya sayfa yenilense dahi veriler kaybolmaz.

-- Silme Özelliği (Sliding): Harcama satırı yana kaydırılarak silme işlemi gerçekleştirilebilir.

-- Hata Kontrolü: Boş veri girişi engellenerek kullanıcıya uyarı verilir.

🛠️ Kurulum ve Çalıştırma
Projeyi yerel makinenizde çalıştırmak için aşağıdaki adımları izleyebilirsiniz:

1. Projeyi bilgisayarınıza indirin veya clone'layın.

2. Terminal üzerinden proje klasörüne gidin.

3. Bağımlılıkları yüklemek için şu komutu çalıştırın:
 npm install

4. Uygulamayı tarayıcıda ayağa kaldırmak için:

   ionic serve
